# clair-frontend

React + TypeScript + PWA 기반 사용자 웹 애플리케이션 저장소다.

## 책임 범위

- 계약서 업로드 화면
- 문서 목록 및 상세 화면
- 분석 결과 대시보드
- 위험 조항, Q&A, 객체 탐지 결과 시각화

## 권장 구조

  app/
  components/
  features/
  lib/
  hooks/
  styles/
