
  # WireFrame

  This is a code bundle for WireFrame. The original project is available at https://www.figma.com/design/ExtopQJzE9xS7XvqNQFTbn/WireFrame.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
